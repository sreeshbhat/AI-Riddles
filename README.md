# AI Vault: Riddles Edition 🧩 (Groq Edition)

A multi-level riddle challenge powered by **Groq**. Solve puzzles to unlock the final vault code.

## Setup Instructions

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Configure API Key**:
   - Ensure you have a `.env` file with `GROQ_API_KEY=your_key`.
3. **Run the App**:
   ```bash
   python -m streamlit run streamlit_app.py
   ```
